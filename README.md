# Research-Oriented University Information System

This project is a console-based OOP model of a research-oriented university.

## Main features
- Authentication
- User roles: Student, GraduateStudent, Teacher, Manager, Admin, TechSupportSpecialist
- Course registration with manager approval
- Marks and transcript
- Research papers and projects
- Journal subscriptions and notifications
- Requests and messages
- Serialization

## Design patterns
1. Singleton — Database
2. Factory — UserFactory
3. Observer — Journal
4. Strategy — ResearchPaper Comparators

## How to run
Compile and run Demo.java.
